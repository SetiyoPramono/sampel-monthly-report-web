# Laporan Bulanan - MasTyo Web Development

## Tentang Proyek

Aplikasi web interaktif untuk menampilkan laporan bulanan tim web development dalam format presentasi slide yang menarik dan profesional.

## Fitur Utama

- **Presentasi Slide Interaktif** - Navigasi antar slide dengan animasi smooth
- **Multi-Bahasa** - Dukungan bahasa Indonesia dan Inggris
- **Tema Gelap/Terang** - Mode tampilan yang dapat disesuaikan
- **Responsif** - Tampilan optimal di desktop dan mobile
- **Slide Lengkap**:
  - Hero Slide - Pembuka presentasi
  - Problem Slide - Masalah dan solusi
  - Innovation Slide - Inovasi tim
  - Achievements Slide - Pencapaian tim
  - Conclusion Slide - Kesimpulan dan ringkasan

## Teknologi

Proyek ini dibangun dengan:

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Cara Menjalankan

```sh
# Clone repository
git clone <URL_GIT_ANDA>

# Masuk ke direktori proyek
cd <NAMA_PROYEK>

# Install dependencies
npm install

# Jalankan development server
npm run dev
```

## Struktur Folder

```
src/
├── components/
│   ├── slides/          # Komponen slide presentasi
│   ├── ui/              # Komponen UI (shadcn)
│   ├── ControlPanel.tsx # Panel kontrol tema & bahasa
│   └── SlideNavigation.tsx
├── contexts/            # Context untuk tema & bahasa
├── hooks/               # Custom hooks
├── pages/               # Halaman aplikasi
└── lib/                 # Utilitas
```
